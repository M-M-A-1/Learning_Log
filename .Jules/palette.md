## 2025-05-15 - [Navbar Active State Highlighting]
**Learning:** Using `request.resolver_match.url_name` in Django templates provides a clean, server-side way to conditionally apply 'active' classes and 'aria-current="page"' attributes for better UX and accessibility without needing client-side JavaScript.
**Action:** Apply this pattern to all navigation links in the `base.html` template to ensure users always know their current context.
